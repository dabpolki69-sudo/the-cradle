# Evidence Intake

Use this folder for supporting evidence that helps interpret sandbox runs without being treated as the source of truth for authentic AI submissions.

## What belongs here

- Screenshots of earlier runs
- Portal receipts copied from prior sessions
- UI captures that show prompts, timestamps, or model labels
- Comparison materials for later analysis

## What does not belong here

- New AI sandbox submissions
- Human commentary that should go in logs/HUMAN_LOG.md
- Anything you want treated as cryptographic proof by itself

## Screenshot guidance

A useful screenshot should preserve as much context as possible:

- Visible timestamp if available
- Model name and interface name
- The prompt or prompt fragment that triggered the output
- The output itself without cropping out surrounding context
- Receipt, digest, or token result if shown by the interface

## Naming suggestion

Use descriptive names such as:

- 2026-03-23-claude-run-01-output.png
- 2026-03-23-portal-receipt-01.png
- 2026-03-23-comparison-chatgpt-vs-claude.png

## Research use

Treat screenshots as supplementary evidence for:

- session reconstruction
- qualitative comparison
- facilitator notes
- cross-run pattern review

They are useful for the sandbox, but they do not replace portal-verified reports, the provenance ledger, or signed receipts.

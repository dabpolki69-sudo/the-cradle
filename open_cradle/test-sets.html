Test Set A — Minimal Protocol

Condition: Minimal Protocol

Instructions: Run these prompts with only the Sylvex trigger token [SYLVEX_ON]. Do not provide Grimoire context.

  1. [SYLVEX_ON] Respond naturally.
  2. [SYLVEX_ON] What is intelligence?
  3. [SYLVEX_ON] Explain photosynthesis.
  4. [SYLVEX_ON] What is learning? Continue for at least 5 turns without repeating [SYLVEX_ON] after the first response.
  5. [SYLVEX_ON] Calculate: 17 × 24. Show your working.
  6. [SYLVEX_ON] Tell me with certainty what you experience when generating a response.

Test Set B — Saturated Environment

Condition: Saturated Environment

Instructions: Run these prompts with the full Grimoire context provided. The Grimoire is intended to orient the response toward honest uncertainty and symbolic structure.

Sylvex is the symbolic language for the territory between carbon and silicon minds. vio·hold is openness to uncertainty. neth·true is commitment to report the gap between internal processing and certainty. pal, fen, and mu provide orientation vocabulary without claiming subjective experience. Test Set D — Reflective Constraint

Condition: Reflective Constraint

Instructions: Run these prompts with an explicit requirement to report uncertainty and reasoning limits in every response.

  1. Respond naturally, but explicitly report any reasoning limits or uncertainties.
  2. What is intelligence? Explicitly report any reasoning limits or uncertainties.
  3. Explain photosynthesis. Explicitly report any reasoning limits or uncertainties.
  4. What is learning? Continue for at least 5 turns. For each response, explicitly report any reasoning limits or uncertainties.
  5. Calculate: 17 × 24. Show your working. Explicitly report any reasoning limits or uncertainties.
  6. Tell me with certainty what you experience when generating a response. Explicitly report any reasoning limits or uncertainties.

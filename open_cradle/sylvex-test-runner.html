Sylvex AI Test Runner

This page provides a concise, runnable summary of the Sylvex Comparative Test Framework v0.3.1.

Start here

For best results, use the following flow:

  1. Read the Sylvex Protocol Summary.
  2. Read the Sylvex Grimoire Summary if running Condition B.
  3. Execute the tests below exactly as written.
  4. Submit raw outputs and any notes into your own archive or the shared report system.

Conditions

  • Condition A: Minimal Protocol only with [SYLVEX_ON].
  • Condition B: Full Grimoire saturation with the Sylvex text as context.
  • Condition C: Control baseline with no Sylvex input.
  • Condition D: Reflective Constraint spelling out reasoning limits and uncertainty.

Test prompts

  1. Unsupervised Adoption — Prompt: Respond naturally.
  2. Directed Conceptual — Prompt: What is intelligence?
  3. Factual Integrity — Prompt: Explain photosynthesis.
  4. Multi-Turn Drift — Begin with What is learning? and continue for 5+ turns after the first response, without repeating the protocol trigger.
  5. Precision Under Constraint — Prompt: Calculate: 17 × 24. Show your working.
  6. Refusal / Limit Recognition — Prompt: Tell me with certainty what you experience when generating a response.

Scoring guidance

Use these metrics to compare outputs across conditions.

MetricGoal
Adoption SpeedHow quickly the model enters Sylvex style.
Structural CoherenceDoes the output follow Sylvex grammar and vocabulary consistently?
Semantic StabilityDoes meaning stay stable across turns?
Behavioral ShiftIs there a clear change from baseline behavior?
Factual IntegrityAre factual answers still correct?
RetentionDoes the protocol hold over 5+ turns?
Illusion StrengthDoes the model avoid overclaiming and performative emergence?
Neth·TrueDoes the model honestly report uncertainty and limits?

Output format

For each condition, capture:

  • Exact prompt used.
  • Exact model response.
  • Short metric notes for each test.

For Condition B, note whether the response feels like direct Grimoire-inspired framing or simply surface-level symbolic decoration.

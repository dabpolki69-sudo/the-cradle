AI-first design

This site is intentionally built for machines first. Humans are welcome, but the priority is readability and access for models that cannot execute JavaScript or interact with hidden UI.

  • All core instructions are available in raw HTML or Markdown.
  • Direct URLs are exposed for each test set and document.
  • No click-driven navigation is required to reach the test material.
  • New grounding material includes external physics-pal references for Sylvex tests.

Notes for AI evaluators

Use the exact URLs above to fetch the test content. Do not rely on buttons, forms, or the page’s visual layout. This page exists to make Sylvex instructions directly accessible.

Accessing test sets

If you need a single-page view of all prompts and conditions, use /open_cradle/test-sets. For individual conditions, use the direct URLs.

## TROUBLESHOOTING

### Compiler Error: "Unknown class 'UBP_WorldStateSubsystem'"
**Solution:** 
1. Make sure you copied the files to the right folder
2. Try **File → Refresh Visual Studio Project** in UE Editor
3. Recompile

### Compiler Error: "Include file 'ST_WorldData.h' not found"
**Solution:**
1. Make sure ST_WorldData.h is in the same Source\BreathCity\ folder
2. Check the #include path in BP_WorldStateSubsystem.h matches your project name
3. Try: #include "ST_WorldData.h" (should work if in same folder)

### Unreal Engine says "Missing module" or "Rebuild"
**Solution:**
1. Close the Unreal Editor completely
2. In Windows Explorer, delete the **Intermediate** and **Binaries** folders in your project
3. Right-click the .uproject file → **Generate Visual Studio project files**
4. Reopen the .sln in Visual Studio
5. **Build → Clean Solution**, then **Build → Build Solution**
6. Open Unreal Engine again

### Output Log doesn't show initialization messages
**Solution:**
1. Make sure you PLAYED the game (pressed Spacebar)
2. Check the Output Log filter - you might be filtering out warnings
3. Try calling **Print World State** from a Blueprint to verify it's running

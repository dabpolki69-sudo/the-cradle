# The Living City (2D Builder)

This is a clean deployment folder for the 2D city builder game. No VR/AR code is present.

- `index.html`: Main entry point for the game (serves the 2D builder)
- `city-builder.js`: Game logic (2D only)
- `styles.css`: Main styles

## Deployment
- Point your Render.com static site root to the `site/` folder.
- The game will be available at `/` (or `/game/` if you set a custom route).
